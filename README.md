FOXTROT-PIT

FOXTROT is a 1.8.9 FORGE MOD FOCUSED TO HELP HYPIXEL PIT PLAYERS WITH A CUSTOM HUD GUI AND ALOT OF USEFUL FEATURES
Features

ALL TOGGLABLE OPTIONS

    ENEMY LIST (EnemyHUD):

    IF AN ENEMY IN YOUR LIST JOINED YOUR LOBBY IT WILL ALERT YOU IN CHAT OF THEM JOINING AND THERE WILL BE A RED BOX AROUND THE PLAYER (ENEMY ESP) IT WILL ALSO SHOW WHAT ITEMS THEY GOT ON AND THEIR LOCATION
